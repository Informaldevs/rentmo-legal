# Data Sharing Addendum

## 1. Purpose and Scope

This Data Sharing Addendum ("Addendum") supplements the Privacy Policy and governs the processing of personal data in accordance with the Data Privacy Act of 2012 (Republic Act No. 10173) of the Philippines.

## 2. Data Processing Activities

### Tenant Information Processing
We process tenant information for the purpose of:
- Facilitating rental agreements
- Managing lease contracts
- Processing rental payments
- Providing property management services

### Landlord Information Processing
We process landlord information for the purpose of:
- Enabling property listing
- Facilitating tenant screening
- Processing rental income
- Providing property management tools

## 3. Legal Basis for Processing

Our processing of personal data is based on:
- Performance of a contract
- Legal obligation
- Legitimate interests
- Consent (where required)

## 4. Data Subject Rights

Under the Data Privacy Act, data subjects have the right to:
- Be informed whether personal data is being processed
- Access their personal data
- Object to processing
- Rectify inaccurate data
- Erase or block personal data
- Damages in case of violation

## 5. Data Security Measures

We implement appropriate security measures including:
- Encryption of data in transit and at rest
- Regular security assessments
- Access controls and authentication
- Employee training on data protection

## 6. Data Retention

Personal data will be retained only for as long as necessary to fulfill the purposes for which it was collected, or as required by applicable laws and regulations.

## 7. International Transfers

Personal data may be transferred to countries outside the Philippines only when adequate safeguards are in place to protect the data.
